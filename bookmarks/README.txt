==============================================
Django Basic Bookmarks
http://code.google.com/p/django-basic-apps/
==============================================

A simple bookmark library application for Django projects.

To install this app, simply create a folder somewhere in
your PYTHONPATH named 'basic' and place the 'bookmarks'
app inside. Then add 'basic.bookmarks' to your projects
INSTALLED_APPS list in your settings.py file.
